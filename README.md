## Запуск

- Переименовать *.env.example* в *.env* и прописать свои параметры подключения
- composer install
- php artisan migrate
- php artisan db:seed --class=UserSeeder
- php artisan db:seed --class=CommentSeeder
- php artisan key:generate
- php artisan serve
- http://127.0.0.1:8000/
